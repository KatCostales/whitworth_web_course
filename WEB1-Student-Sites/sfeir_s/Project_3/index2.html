What is Prof.Dr Style?

Prof.Dr. design uses the simplicity and accessibility of HTML coding and a primitive and practical aesthetic to communicate information. Prof.Dr. design speaks to a very 'human made' style. It came to be in the early 90's which is why it has some principles and aspects that look similar to the tacky yet still basic '90's' design style that we all know and hate...

For a better understanding of this design style and many of the elements that give it it's iconic look, visit this page!

http://contemporary-home-computing.org/prof-dr-style/
