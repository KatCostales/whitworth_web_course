What is parallax scrolling?

Parallax is a type of web design that features animations and backgrounds associated with the user scrolling. This can including a background image staying in place, while the content moves around it, or the background itself moves as the user scrolls. It is a play on optical illusions mainly, when the background moves slower than the foreground because it is "farther away".
